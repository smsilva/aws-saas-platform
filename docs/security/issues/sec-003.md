# SEC-003 — Container image without fixed digest in sample app

| Field | Value |
|---|---|
| **Severity** | Low |
| **Script** | `scripts/08-deploy-sample-app`, line 35 |
| **Status** | Open |

## Problem

The `httpbin` image is referenced without an immutable digest:

```yaml
image: kennethreitz/httpbin
```

## Impact

In a real environment, the implicit `latest` tag may be replaced by a different version between runs, breaking reproducibility and introducing supply chain risks.

## Fix

Pin the image with a SHA256 digest:

```yaml
image: kennethreitz/httpbin@sha256:<digest>
```

For a lab, an acceptable alternative is to use an actively maintained image such as `docker.io/mccutchen/go-httpbin`.
